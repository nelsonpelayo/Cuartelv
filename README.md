# Mapa Interactivo: Las Calles de Moreno Tienen Memoria

## Descripción del Proyecto
Este repositorio contiene el código fuente y los recursos del mapa web interactivo desarrollado para la Dirección General de Proyectos de Hábitat, Vivienda e Integración Urbana del Municipio de Moreno. 

El objetivo de esta herramienta cartográfica es visibilizar, georreferenciar y honrar a las víctimas de la última dictadura cívico-militar a través de las calles del municipio que hoy llevan sus nombres, permitiendo a los ciudadanos interactuar con la información histórica y territorial.

## Tecnologías y Herramientas Utilizadas
El mapa fue construido bajo estándares web modernos y herramientas de código abierto:
* **Procesamiento Espacial y GIS:** QGIS.
* **Motor de Mapas (Librería JS):** OpenLayers.
* **Estructura y Lógica Web:** HTML5, CSS3 y JavaScript (Vainilla).
* **Geocodificación y Búsqueda:** Photon Geocoder.
* **Diseño e Interfaz (UI/UX):** FontAwesome, controles flotantes personalizados y diseño responsivo para dispositivos móviles.
